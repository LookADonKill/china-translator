This is a project that serves two main purpose. It utilized Python to create a simple, yet effective menu. You will be given two choices, to do a live translation or batch translation.

Batch Translation: It's a feature where the code will use Threading to efficiently scoure every lines on the folder. For this code, I used the folder from Kosheng's sensitive-lexicon. Here is the Github Folder Link for that: https://github.com/konsheng/Sensitive-lexicon/tree/main/Vocabulary

Live Translation: It's a feature where the code will ask for two options, either translate from English to China or China to English. Once you've chosen, you may input the wording you would like to translate.

How it Works: This code utilized Google's translate by accessing it through deep translator extension obtained within Python.
